# Fraud Pattern Investigator

A transaction-network analytics tool that surfaces suspicious accounts using behavior-based risk scoring.

## Highlights
- Synthetic account graph + transaction stream generation.
- Risk scoring with volume, imbalance, late-night activity, and round-trip patterns.
- Visual network highlighting flagged entities.
- Ranked suspicious account panel with operational metrics.

## Stack
- HTML5 Canvas
- Vanilla JavaScript analytics
- Responsive CSS dashboard

## Run
Open `index.html` in your browser.
